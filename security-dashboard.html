WARNING Semgrep nginx-host-injection $host used in nginx — Host Header Injection
$host variable can be controlled by attacker request
 frontend/nginx.conf CWE-113 A07:2021 Auth Failures HIGH Use $server_name

Issue

$host reflects the Host header from the request, which an attacker can control. This can lead to cache poisoning or password-reset link hijacking.

Remediation

# Replace $host with $server_name in nginx.conf # $server_name uses your configured server name # and cannot be controlled by the attacker
